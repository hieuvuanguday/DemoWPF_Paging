# DemoWPF_Paging

## Introduction

The **DemoWPF_Paging** project is a WPF application that demonstrates how to implement pagination in a user-friendly manner. This application is designed to help users navigate through large datasets efficiently, displaying a limited number of items per page while allowing users to switch between pages seamlessly.

## Features

- **Pagination**: Implement pagination to manage large datasets effectively.
- **User-Friendly Interface**: Easy navigation with buttons to go to the next, previous, first, and last pages.
- **Dynamic Data Loading**: Load data dynamically as users navigate through pages.

## Technologies Used

- **WPF**: To build the user interface for the desktop application.
- **MVVM Pattern**: To separate the UI logic from the business logic and enhance maintainability.
- **C#**: For application logic.

## Installation

1. **Clone the Repository**: 
    ```bash
    git clone https://github.com/hieuvuanguday/DemoWPF_Paging.git
    cd DemoWPF_Paging
    ```

2. **Open the Project in Visual Studio**: 
    - Open the solution file in Visual Studio.

3. **Run the Application**: 
    - Press F5 or click on the Start button to run the application.

## How to Use

- Navigate through the dataset using the pagination controls at the bottom of the window.
- View details of items by selecting them from the list.
- Use the provided controls to move between pages and refresh the dataset as needed.
